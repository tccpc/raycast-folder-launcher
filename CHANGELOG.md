# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-09

### Added

- Initial release
- Fuzzy search for folders using Fuse.js
- Frecency-based ranking (frequency + recency)
- Support for multiple editors:
  - VS Code
  - Cursor
  - Zed
  - WebStorm
  - IntelliJ IDEA
  - Sublime Text
  - Antigravity
  - Custom applications
- Directory picker for workspace selection
- Configurable search depth (1-5 levels)
- Persistent access history via Raycast LocalStorage
